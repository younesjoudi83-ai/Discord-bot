## Configuration

Définis la variable d'environnement `DISCORD_TOKEN` avec ton token Discord.

- **Railway** : dans Settings → Variables
- **Replit** : dans Secrets

## Commandes

| Catégorie | Commandes |
|-----------|-----------|
| Aide | `!help`, `!help mod`, `!help eco`, `!help fun`, `!help config`, `!help remind` |
| Modération | `!kick`, `!ban`, `!unban`, `!mute`, `!unmute`, `!warn`, `!warns`, `!clear`, `!slowmode`, `!lock`, `!unlock` |
| Économie | `!balance`, `!daily`, `!don`, `!leaderboard`, `!shop`, `!buy`, `!gamble` |
| Fun | `!roll`, `!inventory`, `!8ball`, `!de`, `!coinflip`, `!snipe`, `!avatar`, `!serverinfo`, `!userinfo` |
| Config | `!setautorole`, `!addrr`, `!removerr`, `!addresponse`, `!addword`, `!linkfilter`, `!additem` |
| Rappels | `!reminder 10m Message` |
